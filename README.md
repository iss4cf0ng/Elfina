## Quick Start
```
git clone https://github.com/iss4cf0ng/Elfina
cd ./Elfina
make

./bin/elfina --info <elf_path>
./bin/elfina --mmap <elf_path> [arguments]
./bin/elfina --memfd <elf_path> [arguments] 
```
